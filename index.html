Wykonane Aplikacje
Pogoda app Użyto: JS ES6, Speech Api, Weather Api Aplikacja pobiera lokalizacje oraz dane z zewnętrznego źródła o pogodzie. Google Speech Api odczytuje komendę głosową oraz czyta pogodę. Bro tip: Niezbędne jest włączenie lokalizacji. Bro tip 2: Musi być uruchomiona na Chrome. Github Live
Kalendarz Użyto: JS ES6, jQuery Aplikacja wyświetlająca kalendarz, w którym można dodać planowane wydarzenia. Zapisuje wydarzenia w local storage. Github Live
Tekst edytor Użyto: JS ES6, jQuery Aplikacja edytor tekstu. Pozwala na edycje tekstu i na kilka opcji edytorskich. Aplikacja pozwala także na zapisanie tekstu w local storage lub pobranie go na dysk. Github Live
Music player Użyto: JS ES6, canvas, audio context Aplikacja player muzyczny. Pozwala na odtworzenie wybranych utworów oraz oferuje wizualizację stworzoną za pomocą audio context. Github Live
